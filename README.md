# Dummy Payroll System Project

A simple payroll system dashboard built with HTML, Bootstrap, and JavaScript.

## Features

- Login page with hardcoded credentials (admin/pass123)
- Dashboard with employee overview and list
- White and blue Bootstrap theme

## Files

- `login.html`: Login page
- `index.html`: Dashboard page

## Usage

1. Open `login.html` in a browser.
2. Login with username: admin, password: pass123
3. Access the dashboard.

## Technologies

- HTML5
- Bootstrap 5
- JavaScript
